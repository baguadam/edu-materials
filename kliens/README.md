# Kliensoldali Webprogramozás

A könyvtár a `Kliensoldali Webprogramozás` kurzus segédanyagait tartalmazza.

## Struktúra

| Hol található?          | Leírás                                               |
| ----------------------- | ---------------------------------------------------- |
| classes-custom-elements | JS osztályok, custom elements                        |
| react-starter           | React alapok leírás                                  |
| react-states-routing    | React állapotkezelés, routing                        |
| react-hooks-context     | React hooks, custom hooks, context használata        |
| redux-rtk               | Redux, RTK leírása, store, slice, actions, selectors |
