# Oktatási Segédanyagok

A repo oktatási segédanyagokat tartalmaz az `ELTE` `Webprogramozás` és `Kliensoldali Webprogramozás` kurzusaihoz. A leírások és a kódrészletek célja, hogy segítsék a megértést és példákkal szemléltessék az anyag fontosabb részeit. Ezek sokszor önmagukban nem teljesek, csupán a lényegi részeket tartalmazzák. Előfordulhatnak elírások, pontatlanságok. Ezeknek a javítására, kiegészítésére, vagy bármilyen észrevételre, javaslatra szívesen fogadok `Pull Request`-eket.

## Használat

A repó klónozásához használd a következő parancsot:

```sh
git clone https://github.com/baguadam/edu-materials.git
```
